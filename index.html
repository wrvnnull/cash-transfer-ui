2
Get Source Token Account
curl -s https://api.mainnet-beta.solana.com -X POST \ -H "Content-Type: application/json" \ -d '{"jsonrpc":"2.0","id":1,"method":"getTokenAccountsByOwner", "params":["{YOUR_SOLANA_ADDRESS}", {"mint":"CASHx9KJUStyftLFWGvEVf59SGeG9sh5FfcnZMVPCASH"}, {"encoding":"jsonParsed"}]}'

Copy the pubkey from response.

3
Get Destination Token Account

Same as step 2 but use the recipient address.

curl -s https://api.mainnet-beta.solana.com -X POST \ -H "Content-Type: application/json" \ -d '{"jsonrpc":"2.0","id":1,"method":"getTokenAccountsByOwner", "params":["{RECIPIENT_ADDRESS}", {"mint":"CASHx9KJUStyftLFWGvEVf59SGeG9sh5FfcnZMVPCASH"}, {"encoding":"jsonParsed"}]}'
4
Encode + Send

Encode data with Node.js:

node -e " const buf = Buffer.alloc(10); buf[0] = 12; let amt = BigInt({RAW_AMOUNT}); for(let i=0;i<8;i++){buf[1+i]=Number(amt&0xffn);amt>>=8n;} buf[9] = 6; console.log(buf.toString('base64')); "
